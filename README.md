# Simple-learning-AImodel
A beginner friendly python project that shows how an ai model predicts and corrects the errors , updating the weights.
Concepts Demonstrated                                                                                                          
Basic machine learning intuition                                                                                               
Weights and feature importance                                                                                                 
Prediction and error calculation                                                                                               
Learning rate                                                                                                                  
Iterative model improvement                                                                                                    
Technologies Used                                                                                                              
Python (no external libraries)                                                                                                 
Purpose                                                                                                                         
                                                                                                                                                                                                                                                          
Built to understand the core idea behind how machine learning models learn from mistakes before using advanced AI frameworks.
